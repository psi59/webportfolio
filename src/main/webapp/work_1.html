No 6 Functional Descr

  1. elasticsearch(검색엔진) 적용으로 빠른 검색 + 전문 검색 가능
  2. ajax를 활용하여 리로딩 없이 리스트 추가
  3. 레시피 작성 시 drag & drop으로 파일 추가 및 순서 변경 가능
  4. 레시피 작성 시 임시 저장 가능
  5. 레시피 상세 보기 시 스텝별 타이머 사용 가능, 타이머 종료 시 알람과 함께 텍스트 색상 변경
  6. 새로운 소식이 오면 Push 알림으로 알려드립니다.
